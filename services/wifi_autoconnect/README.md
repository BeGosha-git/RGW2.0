# WiFi AutoConnect Service

Сервис для автоматического подключения к Wi-Fi сети BeRobots.

## Описание

Сервис проверяет подключение к интернету и к сети BeRobots при запуске. Если подключение отсутствует, автоматически подключается к сети. После проверки/подключения сервис уходит в sleep (как Docker контейнер) и периодически проверяет соединение.

## Параметры

- **SSID**: `BeRobots`
- **Пароль**: `Unitree0408`

## Функциональность

1. **Проверка подключения при запуске**:
   - Проверяет наличие интернет-соединения
   - Проверяет подключение к сети BeRobots

2. **Автоматическое подключение**:
   - Если нет интернета или не подключен к BeRobots, пытается подключиться
   - Использует `nmcli` для управления Wi-Fi соединениями

3. **Периодическая проверка**:
   - Каждый час проверяет соединение
   - При потере соединения автоматически переподключается

## Требования

- `nmcli` (NetworkManager CLI) - обычно предустановлен в Ubuntu/Debian
- Права доступа к управлению Wi-Fi (обычно работает без sudo для известных сетей)

## Использование

Сервис автоматически обнаруживается и запускается через `services_manager`. Для ручного запуска:

```bash
python3 services/wifi_autoconnect/wifi_autoconnect.py
```

## Логи

Сервис выводит логи с префиксом `[WiFiAutoConnect]`:
- Статус подключения к интернету
- Статус подключения к BeRobots
- Попытки подключения
- Периодические проверки

## Поведение при ошибках

Сервис обрабатывает все ошибки и не падает. При возникновении ошибки сервис логирует её и продолжает работу в режиме sleep.
